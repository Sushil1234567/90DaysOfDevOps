**Linux Commands Focused on:**

**Process management:**

ps:     Process Status (reports on running processes).

top:    Displays a real-time view of system processes and resource usage.

df:     Disk Free (displays the amount of free disk space).

du:     Disk Usage (estimates file space usage).

kill:   Terminates a process by sending a termination signal.

uname:  Unix Name (prints system information).

uptime: Show system running time and load avarage




**File system:**

ls:    List (lists the contents of a directory).

cd:    Change Directory (navigates between directories).

pwd:   Print Working Directory (displays the current directory's full path).

mkdir: Make Directory (creates a new directory/folder).

rmdir: Remove Directory (deletes an empty directory).

rm:    Remove (deletes files or directories).

cp:    Copy (copies files or directories).

mv:    Move (moves or renames files and directories).

cat:   Catenate (displays and joins file contents).

touch: Creates an empty file or updates the timestamp of an existing file. 



**Networking troubleshooting:**

ping:    Sends data packets to a host to check network connectivity (Packet INternet Groper).

wget:    World Get (retrieves files from the web).

curl:    Client URL (transfers data from or to a server).
